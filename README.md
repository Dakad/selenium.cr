# selenium.cr

Selenium client for interacting with webpages using webdrivers.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     selenium:
       github: matthewmcgarvey/selenium.cr
   ```

2. Run `shards install`
